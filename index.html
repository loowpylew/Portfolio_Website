About PanTrack

What PanTrack does?

Processes the file directory where an individual video is housed.

Processes the file name of the video.

Processes the date/time watermark in the video, and temperature watermark in the video (if present).

Identifies whether the videos are corrupt.

Identifies movement within the video, within a manually adjustable range (the sensitivity is currently set at ….. – although any sensitivity range between….. minimum and ….. maximum is suggested).

All information is automatically loaded within an Excel spreadsheet which can then be further analysed by users using a programming language such as a ‘R’, ‘SPSS’, ‘Python’, or any other software used for statistical analysis.

Why is this useful?

Saves 100's of hours of tedious analaysis and inputting of watermark data and whether or not there is an animal present within a video capture.

Reduces human error i.e. typo when inputting data, miss interperatation of watermark data, identifies movement of likely small rodent in grainy video capture.

Although subject to what you set the sensitivity too, will not completely remove the need to analyse videos. You will still be required to review videos that have captured movement and identify the animal/animals present within video capture.

What you need to know!

Once installation of software is complete, right click on the desktop shortcut and run as administrator.

This is because the program will be stored within 'Program Files' which by default, standard user accounts don't have permission to modify or write files in this directory. This is a security measure designed to protect the integrity of the installed programs and to prevent unauthorized changes that could compromise your system's stability.

When you run an application as an administrator, it allows the program to have elevated privileges, which can bypass these access restrictions and enable you to modify the files within the "Program Files" directory.
